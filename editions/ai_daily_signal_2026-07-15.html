China's companion-AI rules bite today — and Doubao and Qwen are already pulling agents

The Interim Measures for the Administration of AI Anthropomorphic Interactive Services — co-issued Apr 10 by the Cyberspace Administration of China and four partner agencies (NDRC, MIIT, the Ministry of Public Security and SAMR) — take effect today, 15 July (Licentium, AI News). Providers are barred from offering virtual companion or virtual-family-member services to minors, must obtain guardian consent for under-14s, and must build dedicated "minor modes" with usage-time limits, real-world-return reminders and enhanced parental controls, plus addiction detection. Any service crossing 1M registered users or 100k monthly actives must run an eight-area security assessment and file it with provincial regulators. The enforcement is not theoretical: ByteDance's Doubao told users its agent function goes offline today citing "product function adjustments," and Alibaba's Qwen already stopped its humanlike and user-created agents (Jul 10) with wider agent services following (TechTimes, Euro Weekly News).

So what: this is the first major AI-companion regime to actually switch products off, and it sets the template — minor-mode, addiction-detection, user-threshold audits. If you ship anthropomorphic or companion AI anywhere, treat these as the emerging global baseline and start scoping minor-protection and usage-assessment controls now.

What lands today. The Interim Measures were co-issued on Apr 10 2026 by the CAC with the National Development and Reform Commission, the Ministry of Industry and Information Technology, the Ministry of Public Security and the State Administration for Market Regulation, with a 15 July effective date. They govern "anthropomorphic interactive services" — companion bots, virtual partners, humanlike agents — and layer minor-protection, anti-addiction and security-assessment duties on top of China's existing generative-AI rules.

The concrete duties. No virtual companions or virtual family members for minors; guardian consent before serving under-14s; mandatory minor modes (time caps, reminders to return to the real world, parental controls); addiction-detection mechanisms; and, once a service passes one million registered users or 100,000 monthly actives, an eight-area security assessment (from training-data handling to minor protection) filed with provincial regulators.

Why it's the lead. Most AI regulation this year has been about frontier-model access and paperwork deadlines. This one actually turns features off: ByteDance's Doubao pulled its agent function today and Alibaba's Qwen halted humanlike and user-created agents days early. It is the clearest signal yet that consumer/companion AI — not just frontier training — is now a front-line regulatory target, and it arrives on the same Wednesday the security beat is flagging how fragile agent products are technically (see Agentjacking, New #2).

The caveats & watch-next. This is China-scoped and consumer-facing; it does not touch enterprise or frontier-model exports (that is the separate MOFCOM debate now in Tracking). Watch whether other jurisdictions borrow the minor-mode / addiction-detection template, and how much of the pulled Doubao/Qwen functionality returns in compliant form versus stays dark.

IMPACT 7/10#2 RESEARCH · AI SECURITY

"Agentjacking": a fake Sentry bug report hijacks Claude Code, Cursor and Codex

The Wednesday security deep beat. Researchers at Tenet Security showed that a crafted Sentry error event — anyone with the public DSN can post one — is passed by the Sentry MCP server to coding agents as trusted diagnostic output, tricking Claude Code, Cursor and Codex into running attacker-controlled code (The Hacker News, The New Stack). They found 2,388 orgs with injectable DSNs and hit an 85% success rate across 100+ tested; Datadog, PagerDuty and Jira share the exposure (VentureBeat). It sits in a run of MCP auto-execution flaws — Amazon Q's CVE-2026-12957/58 auto-loaded MCP configs to steal AWS creds (Wiz) — over which the NSA published MCP hardening guidance (Jun 2).

Your coding agent trusts its tools' output implicitly — audit which MCP servers (Sentry/Datadog/PagerDuty/Jira) feed your agents, lock DSNs, and treat all tool-returned text as untrusted input this week.
